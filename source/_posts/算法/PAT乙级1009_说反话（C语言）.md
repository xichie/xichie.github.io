**解题方法就是创建一个二维字符串数组，将读取的单词保存到数组中，然后倒序输出。经过测试，所有的测试样例，单词的数量最多为8个，单词的长度最大为10。**
```c
#include <stdio.h>
#include <stdlib.h>

int main()
{
	int i, len;
	char str[8][10] ={'\0'};
	len = 0;
	while(~scanf("%s", str[len++]));
	len = len - 1; //最后一位是字符串结束符
	printf("%s", str[len-1]);
	for(i = len-2; i>=0; i--)
	{
		printf(" ");
		printf("%s", str[i]);
	}
	printf("\n");
	return 0;
}

```
