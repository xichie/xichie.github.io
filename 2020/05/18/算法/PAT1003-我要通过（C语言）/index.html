网上已经有很多解析了，就不废话了。
主要注意一下几点：

  1. P、A、T三个字符都要出现。（测试点2）
  2. P、T仅且出现一次。
  3. P在T之前。
  4. P和T之间至少有一个A。
  5. P左边A的数量 * P和T之间A的数量 = T右边A的数量。

代码简单易懂，就是判断上面几个条件。
